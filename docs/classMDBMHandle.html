Detailed Description

Convenience class to track an MDBM handle.

Convenience class to track and MDBM handle. Wraps all the locking/unlocking functionality for all locking types. Automatically cleans up the handle upon shutdown for smart pointer use. The locks are auto-aquired for the store and delete operations. The lock is auto release when the handle goes out of scope for the handle. Warning the m_kv retains a read lock during read operations until object destruction. Thus, the data should be copied immediately to avoid lock contention.
