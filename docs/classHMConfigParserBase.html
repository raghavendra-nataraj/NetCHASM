Detailed Description

Base class for all configuration parsing types. Derive a new config parser from this class.

Base class for configuraiton parsing. Adds the indirect host support to allow us to map a host check to another host check. Adds the parseDirectory support which will automatically call the appropriate base class parse for each file.
