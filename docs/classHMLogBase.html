Detailed Description

The main base class for the logging.

The main base class for the logging.

All logging types need to be derived from this class and implement the following functions: rotate - do any work needed to re-open file handles after the current log has been rotated out. openLog - setup and open the log file. closeLog - tear down and close the current logging. writeLog - write the log entry to the log.
